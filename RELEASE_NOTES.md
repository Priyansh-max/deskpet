## DeskPet v1.1.0

### ✨ New
- **Settings window** — a tabbed window (open it from the chip menu) for Pet, Readout, Animation, Appearance, Startup, and About.
- **Choose your metric** — CPU %, RAM %, or live network throughput (↓ download / ↑ upload), with an optional label.
- **Load alerts** — the readout pulses and the pet glows when CPU/RAM crosses a threshold (independent CPU & RAM thresholds).
- **Appearance** — transparent or custom-coloured pill, opacity, chip size, and an accent colour.
- **Launch at login**, remember-pause, and hide-under-fullscreen toggles.
- Adjustable **animation speed** (min/max FPS + sensitivity).

### 🐛 Fixes
- Fixed the chip drifting to a random spot after fullscreen apps / resolution / DPI changes — it now stays pinned beside the tray.
- Steadier, symmetric chip that no longer resizes as the numbers tick.

> Not code-signed — SmartScreen may warn on first launch (More info → Run anyway).
